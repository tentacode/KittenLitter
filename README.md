Kitten Litter
=============

This is where I put all my cat crap.

## Prerequisites

* PHP 5.4

## Setup

```
git clone git@github.com:tentacode/KittenLitter.git
cd KittentLitter
composer.phar install
bin/reload
```

## Run

```
app/console server:run
```
